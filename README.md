# English <-> German Trainer

An interactive browser-based trainer for listening, pronunciation, and practical German used in daily life.

Live site:
https://vinodneelakantam.github.io/German_Lang_Learning/

## What is Included

1. Listening chapters split into focused pages (01 to 07)
2. Grammar chapter for present-tense word conjugation (Chapter 08)
3. Click-to-speak vocabulary and sentence playback
4. Role-play scenarios for real-world conversations

## Main Features

1. Two-column English and German listening tables
2. Click any cell to hear speech synthesis
3. Top bar playback controls: Play/Resume, Pause, Stop
4. Speed control: Slow, Normal, Fast
5. Auto-Play-Next topics toggle
6. Role-play playback for scenario-based practice

## Listening Chapter Flow

Listening pages are designed for continuous practice.

If Auto-Play-Next is enabled and you start in Listening 01, playback can continue across listening chapters in order:

1. Listening 01: Core Basics
2. Listening 02: Airport and Conversations
3. Listening 03: Housing and Utilities
4. Listening 04: Work and Jobs
5. Listening 05: Health and Medical
6. Listening 06: Transport and Commuting
7. Listening 07: Role Play Scenarios

This allows one long listening run without manually restarting chapter playback each time.

## How to Use

1. Open the project in a modern browser (Chrome or Edge recommended).
2. Open the home page and choose a listening or grammar chapter.
3. In listening chapters, keep Auto-Play-Next enabled for continuous chapter playback.
4. Use Play/Resume, Pause, Stop, and speed controls as needed.

No build step or server is required.

## Local Setup

1. Clone the repository:
	git clone https://github.com/vinodneelakantam/German_Lang_Learning.git
2. Open index.html in your browser.
