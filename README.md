# English ⇄ German Trainer

An interactive browser-based vocabulary and pronunciation trainer built for immigrants and language learners. No installation needed — open the HTML file or visit the GitHub Pages link.

**Live site:** https://vinodneelakantam.github.io/German_Lang_Learning/

---

## Features

- **1000+ vocabulary entries** covering real immigrant daily-life topics
- **Two-column table layout** — click any English or German cell to hear it spoken
- **Play / Stop controls** per lesson and globally in the top bar
- **Speech synthesis** using the browser Web Speech API (works best in Chrome / Edge)
- **Speed control** — Slow / Normal / Fast
- **Role Play Scenarios** — full practical dialogues you can listen to end-to-end

## Usage

Open `index.html` in Chrome or Edge (speech synthesis requires a modern browser). No server or build step required.

```
git clone https://github.com/vinodneelakantam/German_Lang_Learning.git
# then open index.html in your browser
```
